# RAG-Lab

## 项目说明

本仓库用于承载 RAG 调试平台的需求、原型、设计与后续开发交付物。

当前阶段以“瀑布里程碑 + 敏捷迭代”混合模式推进：

- 瀑布负责阶段关口与评审产物
- 敏捷负责阶段内部的 Sprint 开发与交付

## 当前仓库重点内容

- `docs/`：项目文档区
- `screanshot/prototype/`：原型演示工程
