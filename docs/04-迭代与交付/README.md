# 04 迭代与交付

本目录承接敏捷执行视角，用于管理待办、Sprint 和阶段交付。

建议最少维护：

- 产品待办清单
- 当前 Sprint 计划
- 编码规范
- Sprint 复盘
- 版本交付说明

推荐节奏：

- 每个 Sprint 开始前更新计划。
- 每周更新待办优先级。
- 每个 Sprint 结束后沉淀交付结果和遗留项。
